# MyOptionDiary

## Before You Install

### ⚠️ Windows Security Notice

When you run the installer, Windows may display a security warning saying the software is from an unknown publisher. This is because MyOptionDiary does not yet carry a Windows code-signing certificate — it does not mean the software is harmful.

To proceed, click **"More info"** and then **"Run anyway"**. This is a one-time step and the warning will not appear again after the first install.

---

### ⏱️ First Installation

The first time you install MyOptionDiary, please allow a little extra time for the setup to complete. This is normal — the app is setting up everything it needs to run smoothly on your machine. Grab a coffee and let it finish. It will be ready shortly.
